## Clone and run locally

1. Clone the repo
```
git clone https://github.com/williamKhine/cosa48
```
```
cd cosa48
```

2. Install packages
```
npm i
```

3. Enter Supabase keys
```
mv .env.example .env.local
```

4. Run development server
```
npm run dev
```
